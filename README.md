# DevOps AI Log Analyzer

Application log analyzer built to simplify troubleshooting by identifying common issues and providing possible resolutions from log data.

## Features

- Analyze application logs
- Detect possible root causes
- Classify issue severity
- Generate troubleshooting recommendations
- Dockerized application
- Jenkins CI/CD integration

## Tech Stack

- Python
- Flask
- Ollama
- Docker
- Jenkins

## Future Enhancements

- Kubernetes deployment
- Monitoring dashboard
- Log file upload support
- Security scanning integration
